# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please use GitHub's private vulnerability reporting feature:

1. Go to the repository's **Security** tab.
2. Click **Report a vulnerability** under "Advisories".
3. Provide as much detail as possible (steps to reproduce, affected versions, potential impact).

This keeps the report private while we investigate and prepare a fix, rather than disclosing it
publicly before a patch is available.

We will do our best to acknowledge your report and work with you to understand and address the
issue as soon as possible.
